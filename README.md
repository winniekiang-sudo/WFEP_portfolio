# WFEP — UI/UX Designer Portfolio

A modern Next.js portfolio for showcasing product design work, process, and contact.

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing — hero, selected work, services |
| `/projects` | Full project index |
| `/projects/[slug]` | Case study detail |
| `/about` | Self introduction & background |
| `/contact` | Get in touch |
| `/terms` | Terms & conditions |
| `/privacy` | Privacy policy |
| `/cookies` | Cookie policy |

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

Edit `src/lib/site.ts` and `src/lib/projects.ts` for copy, links, and case studies.
